# Retail-sales-analysis
This project analyses one year of transactional data from a UK-based non-store online retail company that sells all-occasion gift items. The dataset includes both local and international wholesale customers with purchase records from 1st Dec 2010 to 9th Dec 2011.

Key findings:
Certain product categories consistently generated higher revenue.
Repeat wholesalers contributed a significant portion of total sales.
Monthly sales trends showed clear fluctuations, with noticeable seasonal peaks.
A small number of products accounted for a large share of overall revenue.

Tools used:
Python, Pandas, Matplotlib, Colab
